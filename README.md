# Домашні завдання з JavaScript – IT Step

Цей репозиторій містить мої домашні завдання (HW) з курсу JavaScript у коледжі IT Step. Кожне завдання розташоване в окремій гілці або папці відповідно до номера домашнього завдання.

## Структура репозиторію

- `hw-01` – Домашнє завдання 1  
- `hw-02` – Домашнє завдання 2  
- `hw-03` – Домашнє завдання 3  
- …  
- `branches` – Альтернативні гілки з окремими домашніми завданнями
